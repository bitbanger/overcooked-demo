Now that you're familiar with the game, this page will talk a bit about VAL. The ultimate goal of this session will be for you to teach VAL to make onion soup in the game, so let's look at how VAL works.


VAL Basics


VAL takes commands in natural language. If you ask VAL to do something it doesn't know how to, it will ask you to explain it. Your explanation might also contain actions VAL doesn't know yet; VAL will ask about those, too!

VAL starts with a set of "known actions" and allows you to build more complex actions from those. As you can see in the example below, VAL only knows the action "slide", and is taught the action "dance".


When you or I play the Overcooked game, we have to use our arrow keys to move one square at a time. VAL starts with "moveTo" as its movement action, and it knows the names of all of the objects in the environment (plates, pots, the dropoff, etc.) Getting an onion, for example, is two actions for VAL: moving to the onions, and then pressing space.

In the Overcooked setting, your goal is to teach VAL how to cook soup. Although VAL only starts with the actions "moveTo" and "pressSpace", you can still tell VAL to cook the soup right away! Even while explaining an action, don't be afraid to use words and actions that VAL doesn't currently know, because whenever you teach a new one, VAL remembers it so it can use it later.
